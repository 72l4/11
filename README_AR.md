## 📌 **نظرة عامة**

نظام متقدم لإدارة فاتورات العباءات يوفر:
- 📋 إدارة الطلبات (Create, Read, Update)
- 👥 إدارة بيانات العملاء
- 📊 تقارير المبيعات
- 🔍 صفحة تتبع عامة للزبائن
- ☁️ قاعدة بيانات سحابية (Supabase)

**الحالة:** ✅ جاهز للاستخدام

## 🔑 **المميزات الرئيسية**

### ✅ **الإدارة (Admin)**
- إنشاء فاتورة جديدة مع مقاسات تفصيلية
- تحديث حالة الطلب ومعلومات الدفع
- طباعة الفاتورة
- البحث والتصفية
- تقارير شاملة

### ✅ **التتبع (Customer)**
- صفحة عامة بدون تسجيل دخول
- رابط فريد لكل طلب
- عرض حالة الطلب الحية
- مراحل تقدم بصرية
- بيانات محدثة من Supabase

### ✅ **الخصوصية والأمان**
- الزبون يرى بيانته فقط
- لا يمكن التعديل من صفحة التتبع
- Row Level Security في Supabase
- متغيرات البيئة محمية

## 🗄️ **قاعدة البيانات (Supabase)**

### جدول `orders`:
```
خصائص الطلب الرئيسية:
- id (معرف فريد)
- order_no (رقم الفاتورة)
- customer_* (بيانات العميل)
- measurements (6 أعمدة)
- total_amount, paid_amount, remaining_amount
- status (الحالة)
- created_at, updated_at
```

### الفهارس (Indexes):
- `customer_id` - للبحث السريع عن طلبات العميل
- `status` - للتصفية حسب الحالة
- `date` - للترتيب حسب التاريخ

## 🔌 **الاتصال بـ Supabase**

### ملف: `services/supabase.ts`

**الدوال المتاحة:**
```typescript
// إنشاء طلب جديد
createOrder(order: Order)

// تحديث طلب موجود
updateOrder(orderId: string, updates: any)

// جلب طلب واحد (للتتبع)
getOrderById(orderId: string)

// جلب جميع الطلبات
getAllOrders()

// دوال عامة
insertData(table, data)
updateData(table, id, updates)
deleteData(table, id)
```
